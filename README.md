## Instruction

Using `pipenv` instead of `venv`
```bash
# `pip3 install pipenv` doesn't work
# installed via pipx
sudo apt install pipx
pipx ensurepath
pipx install pipenv

# make sure pipenv is installed on system
pipenv --version

# to use Django
pipenv install django

# create Django project
# To activate this project's virtualenv
pipenv shell

```
